Barquinho e Ângulos

Esse sketch foi feito como atividade da disciplina de matemática aplicada à multimídia 1. Nessa atividade nos foi proposto que criássemos uma aplicação que simule o movimento de um barco no mar usando trigonometria.

Para a criação das ondas, utilizei a função seno para fazer o ondulado vertical, para a onda percorrer uma linha horizontal atribuí à bolinha um incremento na posição vertical.

Para o barco, utilizei uma imagem bitmap com o eixo no seu centro utilizando pushMatrix() e popMatrix() para mover o canvas sem interferir nas ondas, entre esse pop e pushMatrix() usei o rotate() com uma função trigonométrica para o barco “oscilar”, porém, nesse ponto o barco ainda estaria parado horizontalmente, para o movimento horizontal o translate foi incrementado horizontalmente, por consequência, o barco.
