Filtros

Esse sketch foi feito como atividade da disciplina de matemática aplicada à multimídia 1. Nessa tarefa criamos filtros para imagens utilizando variáveis para o controle de valores.

Essa foi a imagem utilizada na aplicação:

O primeiro filtro foi o de ruído, para este, utilizei um laço for() para aplicar pixels de outras cores sobre a imagem, nesse caso escolhi 100.000 pixels, ou seja, o laço executa a ação 100 mil vezes, essa ação basicamente é usar uma cor gerada por um random() e aplicar na imagem utilizando a função set().

Resultado:

O segundo filtro foi o cinza, para esse, utilizei dois laços for() para percorrer todos os pixels da imagem, cada vez que um pixel era selecionado usando a função get(), sua cor era coletada, então, usando o modo de cor HBS, eu devolvi para a imagem somente o brightness(brilho) usando a função set().

Resultado:

O terceiro filtro foi de cor vermelha, para esse, novamente utilizei dois laços para percorrer a imagem, porém, para a aplicação da cor vermelha, utilizei o modo de cor RGB, recolhi a cor da imagem original usando a função get() e devolvi somente a quantidade de vermelho de cada pixel usando a função set().

Resultado:

O ultimo filtro foi amarelo, nesse, utilizei basicamente a mesma técnica do filtro vermelho, usando dois laços para percorrer a imagem e usando o modo de cor RGB. A diferença foi que nesse filtro eu recolhi a quantidade de verde e vermelho de cada pixel, pois essa mistura gera amarelo no modo RGB, então para aplicar a cor na imagem fiz uma média aritmética entre os valores de vermelho e verde ((R+G)/2) e apliquei na cor RGB, (((R+G)/2),((R+G)/2),0).

Resultado:
